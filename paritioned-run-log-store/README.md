# Partitioned Run log store provider

This package is an extension to [magnus](https://github.com/AstraZeneca/magnus-core).

## Provides 
Provides capability to have a partitioned run log store.

The extensions of this class are concurrent safe.

## Installation instructions

```pip install magnus_extension_partitioned_run_log_store```

It is not common to install the package by its own as the provided implementation is abstract.
It should be extended to achieve the functionality.
